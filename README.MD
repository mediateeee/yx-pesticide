# 银杏杀虫剂
一款针对某种特定的文件夹病毒 **Windows Explorer.exe** 的查杀工具。

## 制作目的
因大多数杀毒软件都不能很好地都不能很好的查杀这个病毒，故我根据已掌握的信息与deepseek一起编写了这款工具。

## 运行原理
* 设置防护

    通过设置一个空指向的“.symlink”文件并删除所有用户对其的所有权限，以防被删除，以此来达到禁止病毒复制本体的目的。

* 扫描查杀

    扫描选定的目录，查找是否：

    ①有与文件夹名称相同的应用程序，

    ②该文件夹是隐藏的文件夹。

    若两项均满足，则将该应用程序视为病毒。

    该样本还会复制一个本体在根目录下，该文件没有名称；目前还不能解决。

* 检查更新

    利用gitee api，下载最新版本的应用程序。（无论当前是否是最新版本都提示更新。）

## 目录结构
- yx-pesticide.py      - 主程序
- examples             - 本软件所针对查杀的病毒样本
- version.txt          - 用于规定python文件被打包成应用程序时的版本信息

其余的全没有用
