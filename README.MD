# 银杏杀虫剂
一款针对某种特定的文件夹病毒 **Windows Explorer.exe** 的查杀工具。

## 制作目的
因大多数杀毒软件都不能很好地都不能很好的查杀这个病毒，故我根据已掌握的信息借助AI编写了这款工具。

## 运行原理
* 检查电脑

    检查电脑中是否存在该病毒，如存在在删除。

* 扫描查杀

    扫描选定的目录，查找是否有满足以下条件的文件：

    ①有与文件夹名称相同的应用程序，

    ②该文件夹是隐藏的文件夹。

    如两个条件均满足，则视为病毒并查杀。


* 检查更新

    利用 Gitee api ，下载最新版本的应用程序。

## 目录结构
- yx-pesticide.py      - 主程序
- examples             - 本软件所针对查杀的病毒样本
- version.txt          - 用于规定python文件被打包成应用程序时的版本信息

其余的全没有用
